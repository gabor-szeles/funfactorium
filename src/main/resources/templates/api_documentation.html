FUNFACTORIUM API

Here are the paths you can use to query the database for some really fun fun facts!

Path Description
/api/funfact Returns a random fun fact
/api/funfact/{factId} Returns the fun fact with the given ID
/api/funfact/topic Returns you attribute-value pairs of the existing topic ID's and their name
/api/funfact/topic/{topicName} Returns a list of all fun facts in the given topic (case insensitive)
/api/funfact/topic/id/{topicId} Returns a list of all fun facts by the given topic ID

The JSON you receive will contain the following attributes:

Attribute Description
author The user name of the author
topics A list of all the topic names for this entry
rating A one decimal rating between 1 and 5
description The fun fact you were so curious about
id The id of the entry
title The title of the entry
status FOUND - for existing entries.
NOT FOUND - In this case the only other attribute you receive is the description, describing the error
